# Project Context

## Project summary

This repository contains a Godot game that started as an open-world RPG prototype and has now shifted into a darker ARPG / dungeon crawler direction.

The current game direction is:

- mobile-first controls
- dark dungeon-crawler feel
- Diablo-inspired progression and dungeon descent
- authored polish on top of procedural systems

## Current core loop

- start in a village hub
- enter the dungeon
- descend through levels
- fight enemies
- collect loot, gold, and consumables
- improve character power through gear and stats

## Current tool stack

- Godot 4
- VS Code
- GitHub Desktop for git workflow
- Codex for coding and iteration

## How to work efficiently on this project

- Use short, focused chats per subsystem.
- Do not rely on old conversation history as the main source of truth.
- Read the project files in the repo root first.
- Keep changes scoped and testable.

## Current development philosophy

- Fix systems at the source rather than patching around symptoms.
- Favor logical generators and clear rules over fragile visual hacks.
- Keep art readable and stylized rather than overcomplicated.
- Build features in layers: stable backend first, then feel and presentation.
